Simple ATM Machine

The JDBC ATM Machine Project is a console-based application that simulates real-world ATM functionalities by integrating a relational database (e.g., MySQL) using Java Database Connectivity (JDBC). It allows users to perform essential banking operations such as user authentication, balance inquiry, deposits, withdrawals, and viewing transaction history. User data, including account details and transaction logs, are dynamically managed in the database, ensuring real-time updates and secure data handling. This project demonstrates core Java concepts, SQL integration, and database management, making it an excellent addition to a portfolio showcasing practical programming skills.

Global Hosiptal & Blood Bank

The Global Hospital and Blood Bank project is a web-based application designed to streamline hospital management and facilitate seamless access to blood donation and transfusion services. The system provides a user-friendly interface for patients, donors, and administrators, ensuring efficient healthcare delivery and fostering a robust connection between hospitals and blood donors. The project leverages technologies such as HTML, CSS, PHP, and DBMS to create a responsive and dynamic platform. Patients can easily register, book appointments, and access their medical records, while hospitals can manage staff schedules, patient details, and available medical facilities. A dedicated blood bank module allows users to search for blood donors or blood availability in real time, significantly reducing the time required for urgent transfusions.
